Help - File Integrity Monitoring

Welcome to the help section of the File Integrity Monitoring system. Whether you're new to the system or need support, we've compiled key resources to assist you in troubleshooting and understanding the application's features.

Frequently Asked Questions

  • How do I add files to be monitored? – Navigate to the "Home" page, enter the subdirectory name you wish to monitor.
  • How will I be notified of file changes? – The system will send real-time alerts as display notifications in your dashboard whenever a file change is detected.
  • What should I do if I notice unauthorized changes? – Investigate the change by reviewing the file logs. If unauthorized, revert the file to a secure version, and notify your security team for further investigation.

Troubleshooting

If you're experiencing issues, follow these steps to resolve common problems:

  • Verify your internet connection and check that the firewall isn't blocking important communication.
  • Confirm that file paths are entered correctly and are accessible by the system. Missing or incorrect paths can prevent monitoring.
  • Recheck user authentication credentials if you're unable to log in. Ensure your username and password are entered correctly.
  • If the problem persists, clear your browser cache and cookies, or try accessing the application from a different device.
